<h1>Kurs Proje Uygulaması</h1>

<p align="justify">
Proje, .NET 9 ile geliştirilmiş, kullanıcıların kurs alıp satmasını sağlayan, kullanıcılara hitap edecek kapsamlı özellikler barındırmayan ancak mimari ve teknik açıdan çeşitli ve güncel teknolojileri içerisinde barındıran mini bir kurs proje uygulamasıdır. Projede mikroservis mimarisi uygulanmış, servislerde ise genel olarak vertical slice arhitecture yaklaşımı yer almaktadır.

<h3>Teknolojiler ve Yaklaşım</h3>
<h4>
    <ul>Yaklaşım
        <li>Microservice Arhitecture</li>
        <li>Vertical Slice Architecture(Service-based)</li>
        <li>Clean Architecture</li>
        <li>Problem Details (RFC 7807) ile hata yönetimi</li>
        <li>Validation Pipeline (FluentValidation)</li>
    </ul>
    <ul>Tasarım Desenleri
        <li>CQRS Pattern</li>        
    </ul>
    <ul>Teknolojiler
        <li>Asp.Net Core 9 Mvc ve Minimal API</li>
        <li>Ef Core 9</li>
        <li>Docker & Docker Compose</li>
        <li>Fluent Validation</li>
        <li>Automapper</li>        
        <li>API Versioning</li>
        <li>MongoDB</li>
        <li>SQL Server</li>
    </ul>
</h4>


<p>
Proje geliştirilmeye devam etmektedir.
</p>
